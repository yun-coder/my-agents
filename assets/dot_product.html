点积距离 Dot Product

对应位置相乘后求和 —— 关注方向 + 模长的综合度量

📐 核心公式

A · B = a₁×b₁ + a₂×b₂ + ... + aₙ×bₙ
与余弦相似度的关系：A · B = ‖A‖ × ‖B‖ × cos(θ)
点积 = 模长乘积 × 余弦相似度

• 正值大 = 方向一致且模长大  |  0 = 正交  |  负值 = 方向相反
关键区别：受向量模长影响！A=[3,4,1] 和 2A=[6,8,2] 的点积不同

📊 示例：两个 3 维向量

A = [3, 4, 1]     B = [2, 0, 1]

逐步计算过程

维度A 的分量B 的分量逐元素乘积
dim₁323 × 2 = 6
dim₂404 × 0 = 0
dim₃111 × 1 = 1
求和6 + 0 + 1 = 7
A · B = 7

🔄 三种距离度量对比

度量方式公式是否受模长影响典型场景
欧几里得距离 √Σ(aᵢ−bᵢ)² ✅ 是 K-Means 聚类、KNN 分类
余弦相似度 (A·B)/(‖A‖‖B‖) ❌ 否（归一化） 语义检索、文本相似度
点积 Σ(aᵢ×bᵢ) ✅ 是 向量数据库 ANN 检索（加快速度）
为什么向量数据库常用点积？
当所有向量已归一化时（‖A‖ = ‖B‖ = 1），点积 = 余弦相似度，但省去了两次求平方根运算，速度更快。 现代向量数据库（如 Milvus、Qdrant）通常同时支持这两种索引。
