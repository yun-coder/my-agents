余弦相似度 Cosine Similarity

衡量两个向量方向的一致性，与模长无关

📐 核心公式

cos(θ) = A · B / (‖A‖ × ‖B‖) = Σ(aᵢ×bᵢ) / √(Σaᵢ²) × √(Σbᵢ²)
取值范围 [-1, 1]：
• 1 = 方向完全相同（夹角 0°）  |  0 = 互相垂直（无关）  |  -1 = 方向完全相反

📊 示例：两个 3 维向量

A = [3, 4, 1]     B = [2, 0, 1]

逐步计算过程

步骤计算结果
1A · B = 3×2 + 4×0 + 1×1= 7
2‖A‖ = √(3² + 4² + 1²) = √(9+16+1)= √26 ≈ 5.099
3‖B‖ = √(2² + 0² + 1²) = √(4+0+1)= √5 ≈ 2.236
4cos(θ) = 7 / (5.099 × 2.236)= 7 / 11.402
cos(θ) ≈ 0.614 → 夹角 ≈ 52.1° 方向中等偏一致

💡 核心意义

1
关注方向而非大小——向量 A=[3,4,1] 和 2A=[6,8,2] 的余弦相似度为 1
2
文本嵌入中最常用：将文档/句子转为向量后，余弦相似度 = 语义相似度
3
不受向量模长影响：即使用户查询词多词少，方向一致则相似度仍高
