覆盖比例 = 分数
反映信息遗漏风险
评估框架对比 · RAGAS / DeepEval / LLM-as-Judge
框架 特点 API Key 适用场景
RAGAS开源 四大核心指标，无需人工标注，自动计算 OPENAI_API_KEY 标准化离线评估首选
DeepEval开源 支持 RAGAS 指标 + Hallucination + pytest 集成 OPENAI_API_KEY 需要多维度评估 + 测试框架集成
LLM-as-Judge灵活 自定义提示词，任意 LLM 可作裁判 对应 LLM 的 Key 需要定制化评估维度

🔑 需要 API Key

  • RAGAS 框架 (Faithfulness/Relevancy/Precision/Recall)
  • DeepEval 框架 (LLM-as-Judge 评估)
  • LLM-as-Judge 自定义评估

🔓 不需要 API Key

  • 检索指标 (Hit Rate / MRR / NDCG)
  • 传统 NLP 指标 (BLEU / ROUGE)
💡 建议: 使用专用评估 API Key 并设置用量限制，避免生产环境误用
