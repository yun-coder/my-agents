欧几里得距离 Euclidean Distance

空间中两点间的直线距离 —— 最直观的距离度量

📐 核心公式

d(A,B) = √[ (a₁−b₁)² + (a₂−b₂)² + ... + (aₙ−bₙ)² ]
取值范围 [0, +∞)：
• 0 = 两点完全重合  |  值越小 = 距离越近  |  值越大 = 距离越远
注意：受向量模长影响，维度越高时距离容易膨胀（维数灾难）

📊 示例：二维平面上的两点

A = (3, 4)     B = (1, 1)

逐步计算过程

步骤计算结果
1Δx = 3 − 1,   Δy = 4 − 1= 2, 3
2(Δx)² + (Δy)²= 2² + 3² = 4 + 9 = 13
3d = √13= 3.606
d(A,B) = √13 ≈ 3.606

🔄 与余弦相似度的关系

特性欧几里得距离余弦相似度
关注绝对距离（模长 + 方向）仅方向
范围[0, +∞)[-1, 1]
归一化受长度影响自动归一化
典型用途K-Means、KNN、聚类RAG 语义检索、文本相似度
转换公式：如果向量已归一化（‖A‖ = ‖B‖ = 1），则
d²(A,B) = 2 − 2·cos(θ)，即欧氏距离越小 → 余弦相似度越高
