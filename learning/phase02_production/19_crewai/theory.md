# 19 CrewAI

## 核心概念

CrewAI 用 Agent、Task、Crew 和 Process 表达角色型协作：

- Agent：角色、目标和工具。
- Task：任务描述、预期输出和负责人。
- Crew：Agent 与 Task 的集合。
- Process：顺序或层级执行策略。

官方当前建议大型项目优先使用 YAML 配置，便于维护角色与任务。
