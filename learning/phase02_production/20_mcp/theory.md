# 20 MCP Model Context Protocol

## 核心概念

MCP 用统一协议连接模型应用与外部能力。服务端常见能力包括：

- Tools：可执行动作
- Resources：可读取上下文
- Prompts：可复用提示模板

工具不是“自动安全”的。MCP 解决接入标准化，不替代鉴权、审批和审计。
