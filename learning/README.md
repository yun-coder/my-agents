# AI Agent 技术栈系统课程

本目录把技术全景图拆成四个递进阶段。每个知识点包含中文理论文档和带字段注释的 Python Demo。

| 阶段 | 目录 | 核心目标 |
|---|---|---|
| 阶段一 | `phase01_foundation` | Prompt、API、Tool Calling、RAG、基础服务化 |
| 阶段二 | `phase02_production` | 编排、持久化、评估、可观测性、安全与部署 |
| 阶段三 | `phase03_platform` | 类型安全、私有模型、异步任务、多租户与成本控制 |
| 阶段四 | `phase04_frontier` | GUI、多模态与微调数据准备 |

建议严格按阶段顺序学习。默认 Demo 优先使用标准库离线运行；需要模型、浏览器、容器或云服务的示例会显式标注。
